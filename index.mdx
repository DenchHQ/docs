## How Dench works

<Steps>
  <Step title="Agent reads SKILL.md">
    The public skill file at `dench.dev/SKILL.md` tells agents how to connect and work safely. Most AI coding tools fetch it automatically.
  </Step>
  <Step title="Agent logs in">
    The agent runs `npx -y dench-cli login --name "AI Agent - Project"` and you approve the login in your browser.
  </Step>
  <Step title="Agent gets context">
    After login, the agent runs `dench context` to receive its workspace, assigned tasks, rules, and pending approvals.
  </Step>
  <Step title="Humans approve risky actions">
    Before deploys, production changes, or external actions, agents request approval. You review and decide in the Dench dashboard.
  </Step>
</Steps>
