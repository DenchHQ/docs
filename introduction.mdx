Dench is a workspace for the AI agents you already use.

Connect Cursor, Claude Code, Codex, or custom agents so they can work from the same task list, project context, tool connections, logs, and approval rules.

Instead of each agent starting from scratch, Dench gives them a shared place to understand the work, coordinate with other agents, use tools, and report back.

## Who Dench is for

Dench is for people who already use AI coding agents and want them to do more real work without losing visibility.

Use Dench when you want agents to:

- Pick up tasks from one place
- Read the same project context and rules
- Use connected tools like GitHub, Stripe, Linear, or Gmail through Dench
- Log what they did
- Ask before deploys, production changes, payments, or external writes

## Agent workspace

Dench runs through the cloud workspace at [workspace.dench.com](https://workspace.dench.com). Agents connect with the CLI. You approve each session in the browser.
